Pergunta 1: A matemática é uma parte fundamental da programação, e os operadores aritméticos desempenham um papel crucial na realização de cálculos em diversos contextos de desenvolvimento de software. Neste contexto, determine qual das alternativas corresponde ao operador na linguagem Python que é usado para realizar uma divisão e retornando um número inteiro como resultado.
%
//
**
/
:
Gabarito Explicado
Em Python, o operador // realiza a divisão inteira (floor division), descartando a parte decimal.
- / resulta em float (ex: 5/2 = 2.5).
- // resulta em inteiro (ex: 5//2 = 2).
- % retorna o resto da divisão.
Pergunta 2: Pode-se dizer que a linguagem Python não realiza a conversão do tipo da variável automaticamente em suas operações, ou seja, é uma linguagem que para realizar as operações com as variáveis necessitam de conversões do tipo cast, pois ela é classificada como ...
flexivelmente tipada
estaticamente tipada
fortemente tipada
dinamicamente tipada
fracamente tipada
Gabarito Explicado
Python é fortemente tipada porque não permite operações entre tipos incompatíveis (como somar um número com uma string) sem uma conversão explícita (cast).
Nota: Ela também é dinâmica (o tipo da variável é definido na atribuição), mas a pergunta foca na falta de conversão automática, que é característica da tipagem forte.
Pergunta 3 | U012025.2FC-01M: O código utiliza o operador de resto da divisão (%) para determinar o turno em que a máquina está em operação após um certo número total de horas trabalhadas. Considerando total_horas = 50, qual será a saída do programa?
"Indefinido"
"Manhã"
"Tarde"
"Erro de execução"
"Noite"
Gabarito Explicado
Cálculo: 50 % 24 (total de horas no dia).
50 / 24 = 2 com resto 2.
Como o resto é 2, e o intervalo da "Manhã" geralmente compreende o início do dia (0 <= turno < 8), a saída correta é "Manhã".
Pergunta 4: Os comentários em uma linguagem na programação podem auxiliar o programador... determine qual das alternativas corresponde a sintaxe correta de inserir um comentário na linguagem Python.
// Comentário
{ Comentário }
# Comentário
/* Comentário */
<!-- Comentário -->
Gabarito Explicado
Em Python, utiliza-se o caractere # para comentários de uma única linha.
- // é usado em C/C++/Java/JS.
- /* */ é usado para blocos em C/Java.
- <!-- --> é usado em HTML.
Pergunta 5: As estruturas de decisão são importantes na lógica computacional... determine qual das alternativas corresponde a forma correta de utilizar a estrutura de decisão na linguagem Python.
if (x>10)
if: x>10
if x>10 { }
if x>10:
if (x>10) then
Gabarito Explicado
A sintaxe correta do if em Python exige os dois pontos (:) ao final da condição e não utiliza chaves {} nem a palavra then. A indentação na linha seguinte é o que define o bloco de código.
