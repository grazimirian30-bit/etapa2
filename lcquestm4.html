📋 Questionário Comentado - Encontro 4

Questão 1

Qual função é utilizada para abrir um arquivo em Python?

A) read()
B) write()
C) open()
D) file()

✅ Gabarito: C) open()

A função open() é utilizada para abrir arquivos em Python. Ela permite criar, ler ou modificar arquivos dependendo do modo de acesso informado.

arquivo = open("meuarquivo.txt","w")

Nesse exemplo o arquivo é aberto em modo de escrita.

Questão 2

Qual método foi utilizado para gravar informações em um arquivo?

A) save()
B) insert()
C) append()
D) write()

✅ Gabarito: D) write()

O método write() é responsável por escrever dados dentro de um arquivo aberto.

arquivo.write("Olá Mundo")

Após a execução, o texto será armazenado no arquivo.

Questão 3

Qual comando foi utilizado para fechar o arquivo após a gravação?

A) stop()
B) finish()
C) close()
D) end()

✅ Gabarito: C) close()

Depois de utilizar um arquivo é importante fechá-lo. Isso libera recursos do sistema operacional.

arquivo.close()

Esse comando encerra o acesso ao arquivo.

Questão 11

Qual comando foi utilizado para abrir arquivos de forma mais segura?

A) openfile()
B) with open()
C) file.open()
D) readfile()

✅ Gabarito: B) with open()

O comando with open() gerencia automaticamente a abertura e o fechamento dos arquivos.

Isso evita esquecer de utilizar o método close().

with open("dados.txt","r") as arquivo: conteudo = arquivo.read()

Ao finalizar o bloco, o arquivo é fechado automaticamente.

Questão 12

A) write()
B) load()
C) read()
D) split()

✅ Gabarito: C) read()

O método read() retorna todo o conteúdo armazenado dentro do arquivo.

Após a leitura, o conteúdo fica disponível para processamento pelo programa.

A) csv
B) random

✅ Gabarito: C) json

Questão 16

Qual função da biblioteca json grava dados em um arquivo?

A) json.read()
B) json.save()
C) json.dump()
D) json.write()

✅ Gabarito: C) json.dump()

A função json.dump() grava estruturas Python em arquivos JSON.

json.dump(dados, arquivo_json)

Nesse momento os dados são convertidos para JSON.

Questão 17

Qual função da biblioteca json recupera os dados do arquivo?

A) json.load()
B) json.open()
C) json.read()
D) json.get()

✅ Gabarito: A) json.load()

Após abrir o arquivo JSON, utilizamos json.load() para reconstruir os dados.

dados_lidos = json.load(arquivo_json)

O resultado pode ser utilizado normalmente como listas e dicionários Python.

Questão 19

Qual operador foi utilizado para identificar números pares?

A) //
B) **
C) %
D) //

✅ Gabarito: C) %

O operador módulo (%) retorna o resto da divisão.

if num % 2 == 0:

Se o resto for zero, o número é considerado par.
