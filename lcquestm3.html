Questão 1

Qual palavra reservada é utilizada para declarar procedimentos e funções em Python?

A) function
B) create
C) def
D) module

✅ Gabarito: C) def

Segundo o professor, tanto procedimentos quanto funções são declarados utilizando a palavra reservada def.

Após a palavra def informamos o nome do módulo e os parâmetros necessários.

def mostra_nome(): print("Linguagem de Programação")

Esse comando cria um procedimento chamado mostra_nome().

Questão 2

Qual é a principal característica de um procedimento?

A) Sempre retorna um valor
B) Executa uma tarefa específica
C) Cria arrays automaticamente
D) Só pode usar números

✅ Gabarito: B) Executa uma tarefa específica

O procedimento é utilizado para executar uma ação ou conjunto de comandos.

No exemplo apresentado em aula, o procedimento apenas exibia mensagens na tela.

def mostra_nome(): print("Linguagem de Programação")

Observe que não existe return. Por isso ele é considerado um procedimento.

Questão 3

Qual procedimento foi utilizado para exibir traços na tela?

A) mostra_nome()
B) print()
C) tracos()
D) media()

✅ Gabarito: C) tracos()

O professor criou um procedimento chamado tracos() que recebia um número como parâmetro.

def tracos(num): texto = "-" * num print(texto)

Esse procedimento gera uma sequência de traços com a quantidade informada.

Questão 4

Como são chamadas as variáveis recebidas por uma função?

A) Índices
B) Operadores
C) Parâmetros
D) Vetores

✅ Gabarito: C) Parâmetros

Os parâmetros são os dados enviados para uma função ou procedimento durante sua execução.

def tracos(num): print("-" * num)

Nesse exemplo, a variável num é um parâmetro.

Questão 5

Qual comando transforma um procedimento em função?

A) break
B) print
C) input
D) return

✅ Gabarito: D) return

Uma função se diferencia de um procedimento porque devolve um valor para o programa.

Esse retorno é realizado através do comando return.

def soma(a,b): return a+b

Sempre que o Python encontra um return, a função devolve um resultado e encerra sua execução.

Questão 6

Qual era o objetivo da função calcula_media(arr)?

A) Ordenar um array
B) Calcular a média dos elementos de um array
C) Criar uma lista vazia
D) Multiplicar números

✅ Gabarito: B) Calcular a média dos elementos de um array

O professor apresentou uma função chamada calcula_media() que recebia um array como parâmetro.

O objetivo da função era somar todos os elementos do array e dividir pela quantidade de elementos existentes.

def calcula_media(arr): total = sum(arr) media = total / len(arr) return media

Esse tipo de função evita repetir cálculos em vários locais do programa.

Questão 17

B) R$ 160,00

D) R$ 170,00

150 + 15 = 165

Questão 20

Qual será o resultado de 2³ utilizando a função recursiva?

A) 6
B) 8
C) 9
D) 12

✅ Gabarito: B) 8

O cálculo ocorre da seguinte forma:

2³ = 2 × 2 × 2 Resultado: 8

A função realiza chamadas recursivas até que o expoente atinja o caso base.
