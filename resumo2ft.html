1. Conceitos de Ponteiros

Um Ponteiro é uma variável que armazena o endereço de memória de outra variável. Em C, isso permite manipular dados diretamente no hardware.

Operadores Fundamentais

  • & Endereço: Retorna o local da memória onde a variável está guardada.
  • * Conteúdo: Acessa o valor que está no endereço apontado (Deresferência).

Visualização de Apontamento:

0xFF01 42
Variável 'num'
⬅
0xAA05 0xFF01
Ponteiro '*p'
int num = 42;
int *p = # // p recebe o endereço de num
printf("%d", *p); // Saída: 42 (valor de num)

2. Passagem de Parâmetros

A forma como passamos dados para funções define se o valor original será alterado ou não.

Por Valor Por Referência (Ponteiro)
Cria uma cópia isolada na memória. Passa o endereço da variável real.
Alterações NÃO afetam o original. Alterações AFETAM o original.

Aritmética de Ponteiros

Incrementar um ponteiro (p++) faz com que ele aponte para o próximo elemento do tipo declarado:

  • char*: pula 1 byte
  • int*: pula 4 bytes
  • double*: pula 8 bytes

3. Recursividade Avançada

Uma função é recursiva quando resolve um problema chamando a si mesma com uma versão menor do mesmo problema.

Os Dois Pilares:
1. Caso Base: Onde a recursão para (evita o loop infinito).
2. Passo Recursivo: A chamada da função para se aproximar do Caso Base.

Exemplo Visual: Fatorial de 3

fatorial(3) retorna 3 * fatorial(2)
fatorial(2) retorna 2 * fatorial(1)
fatorial(1) retorna 1 * fatorial(0)
fatorial(0) retorna 1 (Caso Base!)
int fatorial(int n) {
    if (n == 0) return 1; // Base
    return n * fatorial(n - 1); // Recursão
}

4. Ponteiro de Segundo Nível

Um ponteiro para ponteiro guarda o endereço de outro ponteiro. É declarado com dois asteriscos (**).
