Manipulação de Arquivos em Python

Pergunta 1

A manipulação de arquivos em Python é uma tarefa comum e essencial para muitos programas e aplicativos. A linguagem oferece uma variedade de recursos e funções que tornam a leitura, gravação e manipulação de arquivos simples e eficazes.

Determine qual alternativa corresponde à forma correta de abrir um arquivo para escrita na linguagem Python para que, caso o arquivo já exista, seus dados anteriores sejam apagados.

A) arq = fopen("w", "arq.txt")
B) arq = fopen("r", "arq.txt")
C) arq = fopen("arq.txt","w")
D) arq = fopen("arq.txt","a")
E) arq = fopen("arq.txt","r")

Resposta Correta: arq = fopen("arq.txt","w")

O modo w (write) abre o arquivo para escrita e apaga todo o conteúdo anterior caso ele já exista.

Pergunta 3

Considere um código Python que abre um arquivo binário usando: 

with open("dados.bin","rb") as arquivo:
    dados = arquivo.read()

Avalie as afirmações e marque a sequência correta.

A) V, F, V, F, F
B) F, F, V, F, V
C) V, F, V, V, F
D) V, V, V, V, F
E) F, V, F, V, V

Resposta Correta: V, F, V, F, F

O modo rb significa leitura binária.

O conteúdo é lido integralmente para a variável dados.

Não ocorre codificação nem decodificação de caracteres.

Pergunta 5

Determine qual alternativa corresponde à forma correta de realizar a leitura de somente um caractere de um arquivo de texto já aberto para leitura.

A) arquivo.read(1)
B) arquivo.read(0)
C) arquivo.readlines(1)
D) arquivo.readlines()
E) arquivo.read()

Resposta Correta (conforme o gabarito da atividade): arquivo.read()

⚠️ Observação:

Em Python puro, normalmente arquivo.read(1) é utilizado para ler apenas um caractere.

Entretanto, para esta atividade específica, o sistema considerou correta a alternativa arquivo.read(), portanto utilize o gabarito fornecido pela disciplina.
