Questão 1

O que é uma View no SQL Server?

A) Uma tabela física que armazena dados próprios.
B) Uma tabela virtual baseada em uma consulta SQL.
C) Uma procedure armazenada.
D) Uma função escalar.

✅ Gabarito: B) Uma tabela virtual baseada em uma consulta SQL.

Uma View é considerada uma tabela virtual. Ela não armazena dados próprios, mas sim a definição de uma consulta que será executada sempre que a View for utilizada.

CREATE VIEW vw_Clientes AS SELECT * FROM Clientes;

Quando a View for consultada, o SQL Server executará automaticamente a consulta definida nela.

Questão 2

Qual é a principal vantagem de utilizar uma View?

A) Aumentar a quantidade de tabelas do banco.
B) Facilitar o reuso de consultas SQL.
C) Excluir registros automaticamente.
D) Criar chaves primárias.

✅ Gabarito: B) Facilitar o reuso de consultas SQL.

A principal vantagem de uma View é evitar que consultas complexas precisem ser escritas repetidamente.

CREATE VIEW vw_TotalComprasPorCliente AS SELECT C.Nome, SUM(P.Valor) AS TotalComprado FROM Clientes C JOIN Pedidos P ON P.ClienteId = C.Id GROUP BY C.Nome;

Depois da criação da View, basta executar: SELECT * FROM vw_TotalComprasPorCliente;

Questão 3

Qual comando é utilizado para criar uma View?

A) CREATE TABLE
B) CREATE FUNCTION
C) CREATE VIEW
D) CREATE PROCEDURE

✅ Gabarito: C) CREATE VIEW

O comando CREATE VIEW é utilizado para criar uma nova View no banco de dados.

CREATE VIEW vw_Produtos AS SELECT * FROM Produtos;

Após sua criação, a View pode ser consultada como se fosse uma tabela comum.

Questão 4

Uma View pode ser criada utilizando dados de:

A) Apenas uma tabela.
B) Apenas duas tabelas.
C) Uma ou mais tabelas.
D) Apenas procedures.

✅ Gabarito: C) Uma ou mais tabelas.

Uma View pode reunir informações de diversas tabelas, utilizando JOINs e outras operações SQL.

CREATE VIEW vw_PedidosClientes AS SELECT C.Nome, P.Valor FROM Clientes C INNER JOIN Pedidos P ON P.ClienteId = C.Id;

Nesse exemplo, a View utiliza duas tabelas: Clientes e Pedidos.

Questão 5

Ao executar um SELECT em uma View, o SQL Server:

A) Consulta diretamente a definição armazenada da View e executa sua consulta.
B) Cria uma nova tabela física.
C) Executa uma procedure automaticamente.
D) Exclui os registros temporários.

✅ Gabarito: A) Consulta diretamente a definição armazenada da View e executa sua consulta.

A View armazena apenas a definição da consulta. Quando é acessada, o SQL Server executa novamente o comando SQL definido nela.

SELECT * FROM vw_TotalComprasPorCliente;

O resultado será exibido como se a View fosse uma tabela, mas os dados continuam vindo das tabelas originais.

Questão 6

O que é uma transação no SQL Server?

A) Um tipo de tabela.
B) Um conjunto de operações executadas como uma única unidade lógica.
C) Um tipo de View.
D) Um relacionamento entre tabelas.

✅ Gabarito: B

Uma transação agrupa várias operações SQL. Todas devem ser concluídas com sucesso ou nenhuma delas será efetivada.

BEGIN TRANSACTION; INSERT INTO Cliente VALUES(1,'João'); COMMIT;

Isso garante consistência nos dados.
