🔄 1. Estruturas de Repetição (Iteração)

A repetição permite que um bloco de instruções seja executado múltiplas vezes até que uma condição seja satisfeita. Em Python, existem duas formas principais:

1.1 O Laço for e a Anatomia do range()

O for é usado para percorrer sequências. A função range() é o motor que gera essas sequências de números [33-35].

1 Argumento range(5) Inicia em 0, vai até 4.
(Saída: 0,1,2,3,4)
2 Argumentos range(2, 6) Inicia em 2, vai até 5.
(Saída: 2,3,4,5)
3 Argumentos range(10, 0, -2) Inicia em 10, decrementa 2.
(Saída: 10,8,6,4,2) 
# Exemplo de Acumulador (Soma de Notas)
soma = 0
for i in range(3):
    nota = float(input("Digite a nota: "))
    soma += nota # Acumulador
print(f"Média: {soma/3:.2f}")

🔁 2. while: Lógica de Sentinela

Diferente do for, o while é ideal para quando **não sabemos** quantas vezes o loop rodará (ex: esperar o usuário digitar uma senha correta) [35-37].

ComandoAção TécnicaUso Comum
breakEncerra o loop completamente.Sair de loops "infinitos" while True.
continuePula o código abaixo e volta para o teste da condição.Ignorar dados inválidos ou números ímpares.
 
# Loop com Break (Simulando do-while)
while True:
    n = int(input("Digite um número positivo (ou negativo para sair): "))
    if n < 0:
        break
    print("Dobro:", n * 2)

Listas são coleções ordenadas de elementos. Elas permitem armazenar um volume massivo de dados em uma única variável [40-42].

🔤 4. Strings: O Array de Caracteres

Textos são arrays imutáveis. O fatiamento (slicing) é uma das ferramentas mais poderosas do Python [43-44].

Sintaxe do Fatiamento: variavel[início : fim]
Nota: O índice "fim" nunca é incluído no resultado. 
texto = "Linguagem Python"
print(texto[0:9])   # Saída: Linguagem
print(texto[10:])    # Saída: Python (até o final)
print(texto[::-1])   # Saída: nohtyP megaugniL (Invertido!)

🏁 5. Arrays Bidimensionais (Matrizes)

Uma matriz é essencialmente uma lista onde cada elemento é outra lista. É a base para planilhas e processamento de imagens [47-50].

[0][0]
[0][1]
[0][2]
[1][0]
[1][1]
[1][2]
[2][0]
[2][1]
[2][2]
Lógica de Acesso:
1º Colchete: Define a Linha.
2º Colchete: Define a Coluna.

Exemplo: matriz[1][2] acessa o valor na 2ª linha e 3ª coluna. 
# Percorrendo uma Matriz com Loops Aninhados
notas = [, ] # 2 alunos, 2 notas cada

for l in range(2): # Percorre linhas
    for c in range(2): # Percorre colunas
        print(f"Nota do aluno {l} na matéria {c}: {notas[l][c]}")

Questão Discursiva 4

Explique como um loop infinito pode surgir em um algoritmo e discuta os impactos desse problema sobre o desempenho de um sistema computacional.

Gabarito: Um loop infinito ocorre quando a condição de parada nunca é atingida.

Explicação: Isso normalmente acontece devido a erros na atualização da variável de controle ou falhas na lógica do algoritmo. Os impactos incluem alto consumo de processamento, desperdício de memória, travamento de aplicações e degradação do desempenho do sistema.
