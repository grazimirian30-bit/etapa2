Questão 1

O SQL Server é classificado como:

A) Uma linguagem de programação utilizada para criar bancos de dados.
B) Um Sistema Gerenciador de Banco de Dados Relacional desenvolvido pela Microsoft.
C) Um sistema operacional para bancos de dados.
D) Um compilador SQL.

✅ Gabarito: B) Um Sistema Gerenciador de Banco de Dados Relacional desenvolvido pela Microsoft.

O SQL Server é um SGBDR (Sistema Gerenciador de Banco de Dados Relacional) criado pela Microsoft. Sua função é armazenar, organizar e manipular dados de forma segura e eficiente.

SQL Server ≠ SQL SQL = Linguagem SQL Server = Software que gerencia o banco

Muitos alunos confundem SQL com SQL Server. O SQL é a linguagem utilizada para conversar com o banco, enquanto o SQL Server é o software que executa esses comandos.

Questão 2

Qual alternativa representa corretamente a diferença entre SQL e SQL Server?

A) SQL é um banco de dados e SQL Server é uma tabela.
B) SQL é uma linguagem e SQL Server é um SGBD.
C) SQL e SQL Server possuem exatamente a mesma função.
D) SQL é utilizado apenas na Microsoft.

✅ Gabarito: B) SQL é uma linguagem e SQL Server é um SGBD.

SQL (Structured Query Language) é a linguagem usada para consultar e manipular dados. Já o SQL Server é o sistema que implementa essa linguagem e gerencia os dados.

SELECT * FROM Cliente;

O comando acima é SQL. Quem executa esse comando é o SQL Server.

Questão 3

Qual subconjunto SQL é responsável pela criação e modificação da estrutura do banco de dados?

A) DML
B) DQL
C) DDL
D) DCL

✅ Gabarito: C) DDL

DDL significa Data Definition Language. É utilizada para definir a estrutura do banco, criando, alterando ou removendo tabelas e objetos.

CREATE TABLE Cliente( Id INT, Nome VARCHAR(100) );

O comando CREATE TABLE pertence ao subconjunto DDL.

Questão 4

Qual subconjunto SQL contém comandos como INSERT, UPDATE e DELETE?

A) DML
B) DDL
C) DQL
D) DTL

✅ Gabarito: A) DML

DML significa Data Manipulation Language. É utilizada para manipular os registros existentes nas tabelas do banco de dados.

INSERT INTO Cliente(Nome) VALUES('João'); UPDATE Cliente SET Nome='Pedro' WHERE Id=1; DELETE FROM Cliente WHERE Id=1;

Todos esses comandos pertencem ao grupo DML.

Questão 5

Qual subconjunto SQL é utilizado para consultas e recuperação de informações?

A) DML
B) DDL
C) DQL
D) DCL

✅ Gabarito: C) DQL

DQL significa Data Query Language. Seu principal comando é o SELECT, utilizado para recuperar dados armazenados.

SELECT * FROM Cliente;

Esse comando retorna todos os registros da tabela Cliente.

Questão 6

Qual tipo de dado é mais adequado para armazenar a idade de um cliente?

A) VARCHAR
B) DATE
C) INT
D) CHAR

✅ Gabarito: C) INT

O tipo INT é utilizado para armazenar números inteiros, ou seja, valores sem casas decimais.

CREATE TABLE Cliente( Idade INT );

Idades normalmente são representadas por números inteiros, por isso INT é o tipo mais adequado.

Questão 7

Qual tipo de dado é indicado para armazenar nomes de clientes?

A) DATE
B) VARCHAR
C) INT
D) FLOAT

✅ Gabarito: B) VARCHAR

VARCHAR é utilizado para armazenar textos com tamanho variável.

CREATE TABLE Cliente( Nome VARCHAR(100) );

O número entre parênteses representa a quantidade máxima de caracteres permitida.

Questão 8

Qual tipo de dado deve ser utilizado para armazenar datas de nascimento?

A) VARCHAR
B) BIT
C) DATE
D) INT

✅ Gabarito: C) DATE

O tipo DATE foi criado especificamente para armazenar datas.

CREATE TABLE Cliente( DataNascimento DATE );

Utilizar DATE facilita consultas e cálculos envolvendo datas.

Questão 9

Qual tipo de dado é mais indicado para armazenar valores monetários com casas decimais?

A) VARCHAR
B) DATE
C) INT
D) DECIMAL

✅ Gabarito: D) DECIMAL

DECIMAL permite armazenar números com precisão, incluindo casas decimais.

CREATE TABLE Produto( Preco DECIMAL(10,2) );

Nesse exemplo podem ser armazenados valores como: 10.50, 1500.99 e 99999.99.

Questão 10

Qual comando DDL é utilizado para criar uma nova tabela no banco de dados?

A) ALTER TABLE
B) INSERT INTO
C) CREATE TABLE
D) DROP TABLE

✅ Gabarito: C) CREATE TABLE

CREATE TABLE é o comando responsável pela criação de tabelas em um banco de dados.

CREATE TABLE Cliente( Id INT, Nome VARCHAR(100), DataNascimento DATE );

Ao executar esse comando, uma nova tabela chamada Cliente será criada no banco de dados.

Questão 11

Qual restrição garante que cada registro da tabela seja identificado de forma única?

A) UNIQUE
B) NOT NULL
C) PRIMARY KEY
D) DEFAULT

✅ Gabarito: C) PRIMARY KEY

A PRIMARY KEY (Chave Primária) é utilizada para identificar de forma única cada registro de uma tabela. Não podem existir dois registros com o mesmo valor de chave primária.

CREATE TABLE Cliente( Id INT PRIMARY KEY, Nome VARCHAR(100) );

Nesse exemplo, cada cliente possuirá um código único. O SQL Server não permitirá dois registros com o mesmo Id.

Questão 12

Qual restrição é utilizada para criar relacionamentos entre tabelas?

A) CHECK
B) FOREIGN KEY
C) UNIQUE
D) DEFAULT

✅ Gabarito: B) FOREIGN KEY

A FOREIGN KEY (Chave Estrangeira) é responsável por criar uma ligação entre tabelas, garantindo a integridade referencial. Ela normalmente referencia uma PRIMARY KEY de outra tabela.

CREATE TABLE Pedido( IdPedido INT PRIMARY KEY, IdCliente INT, FOREIGN KEY(IdCliente) REFERENCES Cliente(Id) );

Nesse exemplo, cada pedido estará associado a um cliente previamente cadastrado.

Questão 13

Qual restrição impede que uma coluna receba valores nulos?

A) UNIQUE
B) CHECK
C) NOT NULL
D) DEFAULT

✅ Gabarito: C) NOT NULL

A restrição NOT NULL obriga o preenchimento da coluna. Se nenhum valor for informado, o banco retornará erro.

CREATE TABLE Cliente( Nome VARCHAR(100) NOT NULL );

Nesse caso, o nome do cliente deverá ser informado obrigatoriamente.

Questão 14

Qual restrição impede a repetição de valores em uma coluna?

A) UNIQUE
B) DEFAULT
C) CHECK
D) FOREIGN KEY

✅ Gabarito: A) UNIQUE

A restrição UNIQUE garante que não existam valores repetidos em uma determinada coluna da tabela.

CREATE TABLE Cliente( Email VARCHAR(100) UNIQUE );

Com essa restrição, dois clientes não poderão possuir o mesmo endereço de e-mail.

Questão 15

A restrição DEFAULT é utilizada para:

A) Criar relacionamentos
B) Definir um valor padrão
C) Impedir valores nulos
D) Criar uma chave primária

✅ Gabarito: B) Definir um valor padrão

DEFAULT permite que o banco atribua automaticamente um valor quando nenhum valor for informado pelo usuário.

CREATE TABLE Cliente( Status VARCHAR(20) DEFAULT 'Ativo' );

Caso nenhum status seja informado durante o cadastro, o SQL Server gravará automaticamente o valor "Ativo".

Questão 16

O comando ALTER TABLE é utilizado para:

A) Criar tabelas
B) Modificar tabelas existentes
C) Excluir registros
D) Consultar dados

✅ Gabarito: B) Modificar tabelas existentes

ALTER TABLE permite alterar a estrutura de uma tabela já criada, adicionando, removendo ou modificando colunas e restrições.

ALTER TABLE Cliente ADD Telefone VARCHAR(20);

Após a execução desse comando, a tabela Cliente passará a possuir uma nova coluna chamada Telefone.
