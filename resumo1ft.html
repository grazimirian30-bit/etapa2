1. Fundamentos e Tipagem em C

A Linguagem C diferencia maiúsculas de minúsculas (Case-sensitive). Variáveis armazenam dados na RAM e constants (#define) permanecem fixas.

Tipos e Formatadores:

TipoExemploFormatador
Inteiroint x = 10;%i ou %d
Real (Float)float y = 3.14;%f
Real (Double)double z = 1.234;%lf
Caracterechar c = 'A';%c
Dica do PDF: O formatador %i pode interpretar bases octais e hexadecimais, diferente do %d.

2. Arrays e Matrizes

Arrays permitem gerenciar grandes volumes de informações do mesmo tipo.

Vetor (Unidimensional)

int v[5] = {10, 20, 30, 40, 50}; // Posições de 0 a 4

Matriz (Bidimensional)

Organizada em linhas e colunas. Em C, a alocação é feita por linhas (Row-major order).

float notas[10][3]; // 10 alunos e 3 notas cada

3. Modularização

O princípio de dividir para conquistar. Bibliotecas comuns incluem math.h e string.h.

  • Procedimento: Usa void e não retorna valor.
  • Função: Usa um tipo (int, float) e exige o return.
float multiplicacao(float a, float b) {
    return a * b; // Retorna o resultado para quem chamou
}

4. Escopo de Variáveis

O local da declaração define a "visibilidade":

  • Local: Dentro de { }. Criada e destruída com o bloco.
  • Global: Fora das funções. Vive durante todo o programa.
  • Parâmetros: Entradas da função, agem como variáveis locais.

Código em C

#include <stdio.h>

int main() {
    int v[20];
    int i, pares = 0;

    for (i = 0; i < 20; i++) {
        printf("Digite o %d valor: ", i + 1);
        scanf("%d", &v[i]);
    }

    printf("\nValores pares:\n");

    for (i = 0; i < 20; i++) {
        if (v[i] % 2 == 0) {
            printf("%d\t", v[i]);
            pares++;
        }
    }

    printf("\nQuantidade de pares: %d\n", pares);

   
}

Definições

#include <stdio.h>
Permite usar funções de entrada e saída como printf e scanf.

int main()
Função principal onde o programa começa.

int v[20]
Vetor com 20 posições para armazenar números.

int i
Variável usada como contador no loop.

int pares = 0
Armazena a quantidade de números pares.

for (i = 0; i < 20; i++)
Estrutura de repetição que percorre o vetor de 0 até 19.

printf
Exibe mensagens na tela.

scanf("%d", &v[i])
Lê um número digitado e armazena no vetor.

v[i] % 2 == 0
Verifica se o número é par (resto da divisão por 2 igual a zero).

pares++
Incrementa o contador de números pares.

return 0
Finaliza o programa corretamente.
